## 特征

- <b>基本：</b>实现不同用户之间的收藏房源，与房东沟通
- <b>多用户：</b>普通用户与管理员各自都能发布房源信息
- <b>开发环境：</b>前端 `Layui`，后端 `Spring` `SpringMVC` `MyBatis`
